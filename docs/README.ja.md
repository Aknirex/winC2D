## 概要

winC2D は Windows 向けのディスク移行ツールです。C ドライブにインストールされたソフトウェアやユーザーフォルダーを別のディスクへ移行し、標準の Windows **シンボリックリンク（Symlink）** とファイルコピーにより C ドライブの空き容量を確保します。アプリケーションのバイナリやレジストリは変更しません。

## 注意事項

ソフトウェアの移行後、元のパスに**シンボリックリンク（Symlink）**を作成し、引き続きアクセスできるようにします。移行後もほとんどのソフトウェアは、アプリ本体やショートカットを変更せずに新しい場所から正常に動作します。**標準の移行操作はレジストリを変更しません。**

> 設定の「デフォルトのインストール先を変更」機能は、新しいアプリのインストール先を変更するため**システムレジストリを変更します**。問題が発生した場合は、設定でデフォルト値に戻すか、システムの復元ポイント / バックアップでロールバックしてください。

## 主な機能

- **インストール済みソフトウェアのスキャン** -- C ドライブのソフトウェアをサイズ・状態付きで表示、複数選択で一括移行
- **ユーザーフォルダー移行** -- ドキュメント・画像・ダウンロードなどのスキャンと移行
- **GUI パス選択** -- ドライブ一覧は自動入力
- **シンボリックリンク** -- 移行後に自動で作成し元パスを維持
- **ロールバック対応** -- 完全な移行ログ付き
- **7 言語対応** -- アプリ内で言語切替
- **ダーク / ライトテーマ** -- システム自動切替、手動変更可
- **自動昇格** -- 起動時に管理者権限を要求
- **Agent 対応** -- AI エージェント・スクリプト向け CLI 付属；[README.ai.md](README.ai.md) を参照

## ダウンロード

1. [Releases](https://github.com/Aknirex/winC2D/releases) から `winC2D-Setup.exe` をダウンロード
2. インストーラーを実行 -- デフォルトで `D:\Program Files\winC2D` にインストール（C ドライブ非占有）
3. GUI、CLI、gsudo 昇格ツールを含み、AI Agent skill を `%USERPROFILE%\.agents\skills\winc2d\` にインストール
4. 移行には**管理者権限**が必要（アプリが自動昇格）
5. コントロールパネル -> プログラムと機能 からアンインストール
6. Windows 10 / 11 対応

## 使い方

1. **スキャン** -- フォルダーを参照し「サイズをスキャン」で容量を計算
2. **選択** -- 移行するフォルダーをチェック
3. **移行** -- フォルダーをターゲットドライブにコピーし、元のパスにシンボリックリンクを作成
4. **ロールバック** -- ログページで完了タスクを選択しロールバック

## Agent CLI モード

AI エージェント・スクリプト向けの CLI については [README.ai.md](README.ai.md) を参照してください。

シンボリックリンクの作成には**管理者権限**または **Windows Developer Mode** が必要です。エージェントの昇格実行には [gsudo](https://github.com/gerardog/gsudo) を使用します。

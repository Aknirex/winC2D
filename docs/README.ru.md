## О программе

winC2D -- инструмент миграции дисков для Windows. Позволяет перенести установленные программы и пользовательские папки с диска C на другой диск, используя стандартные **символические ссылки (symlinks)** Windows и копирование файлов. Бинарные файлы приложений и реестр не изменяются.

## Важные замечания

После миграции программ winC2D создаёт **символические ссылки (symlinks)** по исходным путям, чтобы они продолжали работать. Большинство перенесённых программ запускаются с нового места без изменения самих приложений или ярлыков. **Стандартная миграция не затрагивает реестр.**

> Функция «Изменить путь установки по умолчанию» в настройках **изменяет системный реестр**, чтобы перенаправить установку новых приложений. При возникновении проблем восстановите значение по умолчанию в настройках или откатите изменения через точку восстановления системы / резервную копию.

## Основные возможности

- **Сканирование программ** -- отображение установленных программ на диске C с размером и статусом; множественный выбор и пакетная миграция
- **Миграция папок** -- сканирование и перенос пользовательских папок (Документы, Изображения, Загрузки и др.)
- **Графический выбор пути** -- автозаполнение списка дисков
- **Символические ссылки** -- автоматическое создание после миграции для сохранения исходных путей
- **Поддержка отката** -- полный журнал миграции с откатом в один клик
- **7 языков** -- переключение языка в интерфейсе
- **Тёмная / светлая тема** -- следует за системой, переключается вручную
- **Автоповышение прав** -- запрос прав администратора при запуске
- **Поддержка агентов** -- CLI для AI-агентов и скриптов; см. [README.ai.md](README.ai.md)

## Загрузка

1. Скачайте `winC2D-Setup.exe` со страницы [Releases](https://github.com/Aknirex/winC2D/releases)
2. Запустите установщик -- по умолчанию `D:\Program Files\winC2D` (не занимает диск C)
3. Установщик включает GUI, CLI и gsudo и может связать одну общую копию AI Agent skill с Codex, Claude Code, Antigravity, OpenCode, OpenClaw и другими распространенными агентами
4. Для миграции требуются **права администратора** (программа повышает права автоматически)
5. Удаление через Панель управления -> Программы и компоненты
6. Требуется Windows 10 / 11

## Как это работает

1. **Сканирование** -- перейдите в папку и нажмите «Сканировать размеры»
2. **Выбор** -- отметьте папки для миграции
3. **Миграция** -- winC2D копирует папку на целевой диск и создаёт символическую ссылку по исходному пути
4. **Откат** -- на странице журналов выберите завершённую задачу и нажмите «Откат»

## Режим Agent CLI

winC2D предоставляет CLI для AI-агентов и скриптов. Полная документация: [README.ai.md](README.ai.md).

Создание символических ссылок требует прав **Администратора** или **Windows Developer Mode**. Для повышения прав агентов используйте [gsudo](https://github.com/gerardog/gsudo).
